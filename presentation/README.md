# 交付说明

## 文件清单
- `index.html`：完整 HTML 演示稿，双击即可在浏览器本地打开
- `speaker_notes.md`：与 PPT 每页一一对应的详细讲稿

## 演示方式
- 直接用常见浏览器打开 `index.html`
- 支持 `←`、`→`、`PageUp`、`PageDown`、`Space` 翻页
- 支持 `Home` 跳回首页
- 支持 `F` 进入或退出全屏

## 真实性说明
- 演示稿中的论文、机构、网址、数字均来自公开可检索来源
- 最后一页列出主要参考来源
- 关于系统功能、实施路径与 demo 方案的描述属于项目设计，不代表该产品已在现实世界完成部署

## 本地修改建议
- 所有样式、文案、动画和交互均写在 `index.html` 单文件中，便于直接修改
- 如需替换配色、字体或页面结构，可直接编辑页面内的 `<style>` 与 `<section class="slide">` 内容
