  - 美国（有保留地反对，得分 0.65）：我们认可方案中纳入的适应与损失补偿机制，这是积极一步。但当前案文对气候资金支持的处理缺乏清晰、可预期的安排，这让我们难以接受。我们主张保留一定强度的国际规则约束，确保所有主要排放体真正履行承诺，而不是仅靠自愿行动。
  - 中国（有保留地支持，得分 3.45）：中方对文本中加强技术转移和能力建设、补充适应与损失补偿的安排表示欢迎，这有助于提升发展中国家行动实效。我们支持逐步减少无减排措施煤电，但相关过渡进程需充分尊重各国发展阶段和国情，在灵活安排上不应附带额外限制性条件。中国将继续按照自身NDC承诺，统筹推进能源转型和碳汇能力提升。
  - 巴西（支持，得分 5.95）：巴西欢迎这一政策方向，特别是强化森林保护、碳汇和土地利用治理的安排，这与我国提升减排力度的路径高度契合。我们期待在技术转移与能力建设方面获得切实支持，同时认可为发展中经济体保留灵活过渡空间的做法。
  - 印度（支持，得分 12.7）：作为印度，我们支持加强森林碳汇和适应融资等方向，但文本中“逐步停止新建无减排措施煤电”对我们的能源安全与发展公平构成现实压力。我们希望看到更灵活的阶段性替代方案，并配套充分、可预测的资金与技术转移，以确保我们既能推进非化石能源目标，又不必牺牲基本发展权益。
  - 印度尼西亚（支持，得分 12.0）：我们整体支持分阶段过渡、适应融资与技术转移等安排，这与发展中群岛国家的现实需求一致。但当前方案对化石能源退出强度的处理仍让我们难以接受，希望能针对这一争议点提出更可执行的妥协版本，例如明确退出节奏与技术资金支持的直接对应关系。
  - 沙特阿拉伯（有保留地反对，得分 -1.3）：我们认可方案中对发展中国家分阶段过渡的灵活安排，以及强化森林保护和碳汇等治理举措。但文本中逐步停止新建无减排措施煤电等要求，实际上构成了我们难以承受的化石能源退出强度。我们坚持必须明确保留CCS等过渡技术路线，并细化退出路径，避免一刀切式的冲击。
  - 斐济（支持，得分 11.95）：我们欢迎政策文本中为高脆弱国家提供适应融资与技术转移的安排，这体现了对小岛屿发展中国家特殊脆弱性的认可。斐济到2035年实现能源部门减排36%的承诺，其中24%需要依赖国际支持，因此我们敦促各方确保资金和技术援助充分、可及且及时，以帮助我们筑牢气候韧性根基。
  - 孟加拉国（支持，得分 11.95）：我们欢迎政策纳入对高脆弱国家的适应融资与技术转移支持，这正是孟加拉国实现有条件减排目标的关键前提。我们强调，相关气候资金必须充足、可预测且真正额外于现有援助，才能助力我们这样的最脆弱国家在应对损失与损害的同时推动低碳转型。

  - 印度尼西亚（支持，得分 12.0）：我们感谢文本纳入了分阶段过渡和适应融资，但“逐步停止新建无减排措施煤电”的退出强度对我国仍过于刚性。希望将其调整为更切实的妥协方案：允许在获得充足技术与资金转移的前提下，按各国自主贡献节奏逐步收紧，并同步建立可操作的损失与损害补偿机制，以保障我们这样的群岛国家不掉队。

  - 孟加拉国（支持，得分 11.95）：我们欢迎该政策文本对高脆弱国家的特别关注，尤其是明确承诺为适应融资和技术转移提供支持，这与孟加拉国的核心诉求高度契合。我们强调，实现我们2035年条件性额外减排13.92%的目标，完全取决于能否获得充足、可预测且新增的国际气候资金，以及相应的损失与损害补偿机制。
