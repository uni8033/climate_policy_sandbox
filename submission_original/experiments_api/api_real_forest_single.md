  - 印度尼西亚（支持，得分 16.5）：作为群岛国家，我们支持文本对森林碳汇、适应融资与技术转移的安排，但必须强调，化石能源退出强度与碳边境约束机制若缺乏弹性，将严重挤压印尼的发展空间。我们希望本次推演能针对上述争议点提出更可执行、更体现公正转型的妥协版本，否则我们难以承担过度的调整成本。
  - 沙特阿拉伯（有保留地支持，得分 2.4）：我们支持加强森林碳汇与气候资金安排，这有助于实现共同目标。但当前方案对化石能源退出和碳边境约束的强度仍让我们难以接受。沙特需要在公正转型框架下明确保留CCS等过渡技术路线，并为此获得相应的技术支持与融资。
  - 斐济（支持，得分 15.7）：斐济支持这项文本，其中明确的适应融资、损失损害支持与技术转移机制正是我们这样的脆弱小岛屿国家亟需的保障。我国减排承诺中有相当比例依赖国际援助，因此我们期待这些条款能够尽快转化为可及的资金与技术支持，并认可对发展中经济体保留更长过渡期的安排。
  - 孟加拉国（支持，得分 15.7）：孟加拉国支持这一平衡的政策方向，特别是其中对适应融资和损失损害机制的强化。作为最易受气候影响的国家之一，我们强调：任何森林碳汇和土地利用治理措施，都必须配有充足、可预测且额外的资金与技术转移，才能真正帮助我们实现条件性减排目标并保障社区韧性。
