## 六阶段谈判流程

### 立场陈述
- 阶段说明：各角色先依据自身约束条件给出初始表态。
- 当前政策属性：
  - 化石能源退出强度：3
  - 碳边境约束强度：3
  - 气候资金支持：0
  - 技术转移支持：0
  - 过渡期灵活性：0
  - CCS 过渡技术支持：0
  - 适应/损失补偿支持：0
  - 森林/LULUCF 支持：0
- 角色表态：
  - 发达国家联盟（支持，得分 9.5）：我们支持2030年前停止新建无减排措施煤电，并对高碳产品加强碳边境约束，这体现了以更高标准推动全球转型的必要雄心。为确保政策有效落实，我们主张将技术转移和能力建设支持纳入框架，帮助各方提升减排能力，同时建立透明、可核查的配套机制，使责任传导真正落地。
  - 大型新兴经济体（强烈反对，得分 -7.7）：我们坚决反对当前方案中对碳边境约束和化石能源退出强度的处理方式，这些单边措施严重侵蚀我们的发展权和政策空间，无视大型新兴经济体在工业化、能源安全与产业升级中的现实压力。在缺乏明确、可量化的资金支持、技术转移承诺以及更长的分阶段过渡安排之前，我们无法接受任何此类强制性约束。
  - 最不发达国家集团（有保留地反对，得分 -0.5）：我们支持更明确地限制高排放能源扩张，但当前方案中的碳边境约束强度对我们来说负担过重、缺少公平过渡的保障。若没有更明确的资金支持承诺、技术转移和能力建设安排，以及损失与损害相关的适应补偿，我们很难接受这一案文。
  - 小岛屿与高脆弱国家联盟（支持，得分 9.0）：我们支持这一政策方向，限制无减排煤电和加强碳边境约束有助于推动全球实质减排，但仅靠减缓行动远远不够。必须同步建立可执行的损失与损害补偿机制，并明确提供充足、可预测的适应资金，否则对高脆弱国家而言，生存危机仍无法解除。
  - 化石能源出口国集团（强烈反对，得分 -10.4）：我们强烈反对目前方案中对化石能源退出和碳边境约束的激进安排，这直接威胁出口收入和财政稳定，并将导致严重的资产搁浅。我们要求加入更长的过渡期和分阶段路径，并明确保留碳捕集与封存（CCS）等过渡技术路线，避免一刀切式冲击。
  - 清洁技术与低碳产业联盟（支持，得分 6.0）：我们欢迎这一明确的政策信号，限制无减排措施煤电并强化碳边境约束，将为清洁技术投资和产业化规模化提供稳定的市场预期。为充分发挥全球减排潜力，我们建议在此基础上增加技术转移和能力建设支持，帮助更多地区获取低碳解决方案，实现广泛而公正的转型。
  - 多边开发金融机构（有保留地支持，得分 2.5）：这一政策方向有助于明确高排放能源的退出预期，为项目规划和融资决策提供重要信号。但从可实施性出发，我们建议同步增加更明确的资金支持与技术转移承诺，并设计更长的过渡期和分阶段安排，以确保转型平稳且可融资。

### 冲突识别
- 阶段说明：初始表态阶段判断：主要冲突集中在碳边境约束强度、化石能源退出强度；建议优先处理：弱化化石能源退出强度、提高气候资金支持、提高技术转移支持、提高过渡期灵活性、提高CCS 过渡技术支持。
- 主要冲突：碳边境约束强度, 化石能源退出强度

### 联盟与分裂
- 阶段说明：根据共同诉求识别潜在联盟与关键摇摆方。
- 潜在联盟：
  - 高雄心联盟：发达国家联盟, 最不发达国家集团, 小岛屿与高脆弱国家联盟, 清洁技术与低碳产业联盟, 多边开发金融机构
- 关键摇摆方：
  - 最不发达国家集团：有保留地反对。立场处在中间地带，可能成为交换条件中的关键摇摆方。
  - 多边开发金融机构：有保留地支持。立场处在中间地带，可能成为交换条件中的关键摇摆方。

### 条件交换
- 阶段说明：条件交换阶段围绕 弱化化石能源退出强度、提高气候资金支持、提高技术转移支持、提高过渡期灵活性、提高CCS 过渡技术支持 展开；主要需要在 高雄心联盟 之间达成最小可接受交换。
- 主要交换条件：
  - 用更明确的资金与适应安排交换更高减排接受度
  - 用过渡期和技术路线缓冲高冲击退出要求

### 最终表决
- 阶段说明：最终表决阶段判断：主要冲突集中在碳边境约束强度、化石能源退出强度、过渡期灵活性；建议优先处理：弱化化石能源退出强度、提高过渡期灵活性、提高CCS 过渡技术支持。
- 当前政策属性：
  - 化石能源退出强度：2
  - 碳边境约束强度：3
  - 气候资金支持：1
  - 技术转移支持：1
  - 过渡期灵活性：1
  - CCS 过渡技术支持：1
  - 适应/损失补偿支持：0
  - 森林/LULUCF 支持：0
- 角色表态：
  - 发达国家联盟（支持，得分 8.6）：我们支持在2030年前停止新建无减排措施的煤电，并以更严格的碳边境约束推动减排责任传导，这是全球低碳转型的关键一步。但我们认为，当前方案对“过渡期灵活性”的模糊处理可能削弱规则的约束力，必须设定明确时限和可核查的透明度机制。我们愿意同步提供技术转移和能力建设支持，但这需要与清晰、可核查的减排路径挂钩，以确保高标准真正落地。
  - 大型新兴经济体（有保留地反对，得分 -1.3）：我们理解并认同全球减排的迫切性，但当前方案对“碳边境约束强度”和“化石能源退出强度”的处理，未充分顾及我们这样的大型新兴经济体的发展权与能源安全需求，因此难以全然接受。我们要求加入更长的、有明确分阶段安排的过渡期，并配套可落地的技术转移、能力建设与资金支持，以确保能在产业升级与减排之间保留必要的政策空间。
  - 最不发达国家集团（支持，得分 4.0）：我们支持逐步减少无减排措施的煤电，但当前方案中的碳边境约束强度可能严重冲击我们脆弱的经济，这让我们难以接受。为此，我们呼吁加入更明确的资金支持承诺，并提供损失与损害补偿、技术转移和能力建设安排，以确保气候行动不会以牺牲最脆弱群体为代价。
  - 小岛屿与高脆弱国家联盟（支持，得分 8.2）：我们支持对高排放能源的更严格约束，但必须指出，方案中关于“过渡期灵活性”和“CCS技术支持”的漏洞会严重稀释减排力度，这对我们这些既脆弱又缺乏适应能力的国家而言是不可接受的。要想争取我们的持续支持，必须同步加入明确的损失与损害补偿安排和可执行的适应资金承诺，确保责任传导的同时不让我们独自承受气候代价。
  - 化石能源出口国集团（强烈反对，得分 -4.4）：我们理解气候行动的紧迫性，但当前方案对“化石能源退出强度”和“碳边境约束强度”的规定过于激进，将严重冲击能源出口国的财政稳定并加剧资产搁浅风险。我们呼吁加入更长的分阶段过渡安排，保留CCS等过渡技术作为合规选项，并细化退出路径，避免一刀切式的脱碳要求。
  - 清洁技术与低碳产业联盟（支持，得分 7.7）：我们欢迎这一明确的政策走向：更强的边境约束能有效传导减排责任，限制无减排措施煤电则为低碳产业释放长期市场空间。希望后续配套中加强技术转移与能力建设支持，助力清洁技术更快实现规模化部署与全球扩散。
  - 多边开发金融机构（支持，得分 6.6）：我们支持这一政策方向，但建议将其与更明确的资金承诺和技术转移安排挂钩，确保转型补偿和气候融资有可操作的路径。同时，请考虑为煤电退出提供分阶段的过渡期，以便我们配合项目落地并撬动更多私营资本参与。
