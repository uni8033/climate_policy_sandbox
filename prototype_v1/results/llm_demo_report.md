# 气候政策多利益相关方博弈沙盘 Prototype 1.0

## 输入政策

- 标题：样例政策：2030年前停止新建无减排措施煤电，并提高碳边境约束
- 文本：2030年前停止新建无减排措施煤电，并对高碳产品提高碳边境约束。
- 初始属性：
  - 化石能源退出强度：3
  - 碳边境约束强度：3
  - 气候资金支持：0
  - 技术转移支持：0
  - 过渡期灵活性：0
  - CCS 过渡技术支持：0
  - 适应/损失补偿支持：0

## 三轮推演记录

### 第 1 轮
- 本轮政策属性：
  - 化石能源退出强度：3
  - 碳边境约束强度：3
  - 气候资金支持：0
  - 技术转移支持：0
  - 过渡期灵活性：0
  - CCS 过渡技术支持：0
  - 适应/损失补偿支持：0
- 各角色表态：
  - 发展中国家代表（强烈反对，得分 -8.6）：当前方案在碳边境约束强度和化石能源退出强度上的要求，完全无视发展中国家的发展阶段和现实困难，实质上是将减排压力单方面转嫁给我们，严重威胁能源安全与脱贫进程。我们强烈要求纳入更明确的资金支持承诺、更长的分阶段过渡期，以及切实的技术转移和能力建设安排，否则这样的方案我们无法接受。
  - 发达国家联盟代表（支持，得分 9.9）：作为发达国家联盟，我们欢迎这一明确限制无减排措施煤电扩建的步骤，并认为强化的碳边境约束机制有助于将减排责任更有效地传导至高碳产品供应链。我们呼吁在此框架下建立严格的透明核查体系与统一国际规则，确保这些措施得到迅速且可验证的落实。
  - 传统能源/重工业代表（强烈反对，得分 -9.4）：我们强烈反对当前方案。要求一刀切在2030年前停止新建无减排措施煤电，既未考虑能源安全和就业冲击，也忽视了资产搁浅带来的巨大经济代价；同时碳边境约束的过快收紧将直接削弱我们的国际竞争力。我们要求加入更长的过渡期和分阶段安排，并明确保留CCS等过渡技术路线，以务实而非激进的方式推进转型。
  - 小岛国/环保 NGO 代表（支持，得分 8.2）：我们支持2030年前停止新建无减排措施煤电并提高碳边境约束，这是限制高排放能源扩张、推动责任传导的重要一步。但从气候正义出发，必须同步纳入损失与损害或适应补偿安排，并明确资金支持承诺，否则最脆弱的小岛国和社区仍将承受不成比例的代价。
- 观察员总结：第 1 轮观察员判断：当前主要冲突集中在 化石能源退出强度, 碳边境约束强度。 建议下一轮优先调整：弱化碳边境约束强度、提高气候资金支持、提高技术转移支持、提高过渡期灵活性、提高CCS 过渡技术支持。
